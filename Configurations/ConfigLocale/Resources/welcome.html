ConfigLocale

No payload, its just a script to set the default Locale.

Default system locale:
/Library/Preferences/.GlobalPreferences.plist

Default system keyboard input:
/Library/Preferences/com.apple.HIToolbox.plist

Default login window keyboard input:
/var/tmp/Library/Preferences/com.apple.HIToolbox.loginwindow.plist

User locale:
$HOME/Library/Preferences/.GlobalPreferences.plist

User keyboard input:
$HOME/Library/Preferences/com.apple.HIToolbox.plist
